### mvn profile 简介
#### single 启动使用
![img.png](img/img.png)
1. 维护 `src/main/resources/application.properties`
2. 进入 src/main/java/com/xdap/fruit/FruitDemoApplication.java 点击启动
#### lib 打包使用
![img.png](img/img-lib.png)
1. 如图所示，点击 ▶️ 按钮，进入项目target目录即可获取到待上传到aPaas平台的自开发包
